El adaptador tiene por objetivo acoplar interfaces incompatibles.

Se basa en el principio de composicion de objetos: el adaptador implementa la interface de un objeto y envuelve el otro.
un ejemplo clasicp de adaptador es cuando convertimos un Object[] en un List<Object> Arrays.asList(), List.of(), etc.

El adaptador hereda una clase compatible con el cliente y hace agregacion del objeto a adaptar,
y modifica el comportamiento de los metodos del objeto agregado para hacerlo compatible con el codigo cliente ya definido.
